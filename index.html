Webserv Test Bench

Use this page to verify your server's routing, method handling, and parsing mechanics.

1. Test GET Method & Query Strings

Tests basic file retrieval and your server's ability to extract query parameters.

2. Test standard POST (application/x-www-form-urlencoded)

Tests your basic request body parsing logic.

3. Test File Upload (multipart/form-data)

Tests your stream-reading implementation, boundary parsing, and the extraction of the filename attribute.

4. Test DELETE Method

Tests your server's capability to safely delete a file from the root or target upload directory.

5. Test Error Triggering

Verify that your server correctly serves designated error pages instead of crashing.

  • Trigger 404 Not Found
  • Trigger 403 Forbidden (If index/autoindex is disabled)
